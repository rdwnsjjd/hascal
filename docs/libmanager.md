# Hascal Library Manager(HLM)

**HLM** is default and builtin Hascal's library manager.

## Install a library
```
$ hascal install <library name>
```

example :
```
$ hascal install log
```

## Uninstall a library
```
$ hascal uninstall <library name>
```

example :
```
$ hascal uninstall log
```
